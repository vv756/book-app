# Book App

A simple web application for displaying and managing book information.

This project was created to practice React fundamentals and front-end development concepts.

## Features

- Display a list of books
- Simple UI for browsing book data
- Component-based structure using React
- Responsive layout

## Technologies Used

- React
- JavaScript
- HTML5
- CSS3

## Installation

Clone the repository:

git clone https://github.com/vv756/book-app

Navigate to the project folder:

cd book-app

Install dependencies:

npm install

Run the project:

npm run dev
